# Plonk

## Abstract

We implemented the plonk library as a pallet in order for developers to customize circuits and use the plonk protocol on Substrate runtime. This project allowed us to use following functionalities.

- Custom circuit
- Trusted setup
- Generate proof
- Verify proof
